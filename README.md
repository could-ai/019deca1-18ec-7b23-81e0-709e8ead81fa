# SmartBite PK 🍔🤖

SmartBite PK ek AI Voice Order Taking app aur dashboard hai jo khas taur par Pakistan ke restaurants ke liye design kiya gaya hai. Yeh system khud-ba-khud customers ki calls pick karta hai, unse natural Urdu/English mein order leta hai, aur restaurant owner ke dashboard par automatically bill aur order details generate kar deta hai.

## 🚀 Product Overview

Pakistan mein rush hours ke dauran aksar calls miss ho jati hain, jiski wajah se restaurant ka kafi revenue zaya ho jata hai. SmartBite AI receptionist is maslay ko hal karta hai:
- **No Missed Calls:** AI har call attend karta hai.
- **Smart Voice AI:** Customer se baat cheet karta hai (Roman Urdu, Urdu, English).
- **Auto-Billing:** Order confirm hotay hi seedha dashboard par bill ready ho jata hai.
- **Thermal Print Ready:** 1-click se receipt printer par nikal aati hai.

## 🛠️ Tech Stack
- **Frontend / Dashboard:** Flutter (Web, Windows, Android, iOS)
- **Backend AI Engine (Planned):** Node.js / Supabase Edge Functions
- **Telephony & Voice AI (Planned):** Twilio / Vapi / OpenAI Realtime API
- **Database (Planned):** Supabase (PostgreSQL)

## 📱 Core Features (MVP)
1. **Live Order Dashboard:** AI jo calls receive kar raha hai uski live transcription aur status.
2. **Order Management:** Pending, Confirmed, Preparing, Delivered.
3. **Auto-Bill Generation:** GST aur delivery charges ke sath 1-click bill generation aur printing.
4. **Customer Info Memory:** Puranay customers ki details (address, number) yaad rakhta hai.

## 🏃‍♂️ Setup & Run (Local Development)

Is project ko run karne ke liye aapke paas Flutter SDK install hona zaroori hai.

```bash
# Repo clone karein
git clone <repository_url>

# Dependencies install karein
flutter pub get

# App run karein (Web ya Desktop par behtareen view ke liye)
flutter run -d chrome
```
